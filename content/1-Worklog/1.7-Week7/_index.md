### Week 7 Objectives:

* Understand AWS system observability and performance monitoring using Amazon CloudWatch.
* Design diagnostic dashboards monitoring critical resource utilization metrics (CPU, RAM, Network).
* Establish Amazon Simple Notification Service (SNS) messaging rules and tie them to CloudWatch Alarms.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | --- | --- | --- | --- |
| 2 | **Cloud Observability Study:**<br>- Study Amazon CloudWatch core features: metrics, dimensions, namespaces, alarms, log groups, and diagnostic log streams.<br>- Learn metric resolution rates and default server metrics provided by hypervisors. | 01/06/2026 | 01/06/2026 | [FCAJ Cloud Journey](https://cloudjourney.awsstudygroup.com/) |
| 3 | **Custom Dashboard Configuration:**<br>- Design and configure a customized CloudWatch Dashboard grouping key graphs: EC2 CPU Utilization, Network In/Out traffic, and Application Load Balancer Target Response Times. | 02/06/2026 | 02/06/2026 | [AWS CloudWatch Logs Core Concepts (YouTube)](https://www.youtube.com/watch?v=HRJnhzSSFtk) |
| 4 | **Notification Topic Provisioning:**<br>- Create an Amazon SNS Topic for routing emergency platform notifications.<br>- Create an email Subscription endpoint to the topic and verify the connection through verification links sent by AWS. | 03/06/2026 | 03/06/2026 | [Amazon SNS Developer Guide](https://docs.aws.amazon.com/sns/latest/dg/welcome.html) |

### Week 7 Achievements:

* **Observability Foundations:** Implemented cloud infrastructure observability using Amazon CloudWatch metrics.
* **Unified Metrics Dashboard:** Created monitoring views displaying critical hardware statistics, simplifying diagnostic audits.
* **Automated Alert Pipelines:** Built real-time notification flows using Amazon SNS email integrations.
* **Early Anomaly Detection:** Configured alarms to flag resource issues before they affect end-user service quality.
* **End-to-End Flow Validation:** Verified system response flow from metric triggers to inbox notification.
