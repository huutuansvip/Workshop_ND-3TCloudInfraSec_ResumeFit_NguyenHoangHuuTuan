This lab provides detailed instructions on how to comprehensively deploy the **Resume Fit** system on the AWS cloud platform. Resume Fit is an AI application that helps evaluate the match between candidate Resumes (CV) and Job Descriptions (JD) quickly and transparently to support recruitment decisions.

The system is designed with a standard AWS architecture, ensuring scalability and high availability, including core components: Virtual Private Cloud (VPC), Elastic Compute Cloud (EC2, Auto Scaling Group), Relational Database (RDS PostgreSQL), Object Storage (S3), Asynchronous Processing Queue (SQS), Monitoring Service (CloudWatch), and Secrets Management Service (Secrets Manager).
